GitHub Stats Bot Command

This command retrieves and displays GitHub statistics for a specified user. It provides details such as the number of repositories, stars, forks, followers, following, contributions, and top languages used by the user. The output is presented in a visually appealing format.

User Profile Card Command

This command generates a profile card for a specified user, displaying their details such as name, user ID, gender, and vanity URL. The output is presented in a visually appealing format.

Palestine Frame Command

This command generates a personalized frame with the Palestinian flag and a supportive message for Palestine. The frame includes the user's profile picture, along with a hashtag or message promoting peace and justice for Palestine.
